# Love-in-Progress

一個使用 Python Tkinter 製作的趣味互動視窗程式，它能在您的螢幕上動態排列出心形視窗，並伴隨隨機的溫馨問候與關懷字句，為您或您關心的人帶來互動體驗。

## 快速開始

### 系統需求

- Python 3.x
- Tkinter 模組（Windows 上的 Python 安裝包已預設包含；Linux 用戶可能需要透過套件管理員安裝，例如 `sudo apt-get install python3-tk`）

### 執行步驟

請在終端機中執行以下命令以啟動程式：

```bash
python Love-in-Progress.py
```

### 操作說明

- 啟動後，程式會先以小視窗逐步在螢幕上圍繞出一個心形。
- 接著，會在螢幕隨機位置快速產生大量的溫馨問候視窗。
- **緊急退出**：在程式執行期間，按下 **空白鍵 (Space)** 即可立刻關閉所有彈出視窗並退出程式。
- 若無人為干涉，所有彈出視窗將在一段時間後自動依序關閉並結束程式。

## 功能特點

- **動態心形排列**：利用心形數學極座標公式，計算出螢幕上的相對位置，並依序建立定位視窗。
- **溫馨關懷語錄**：視窗標籤會隨機顯示預設的貼心字句，如「好好愛自己」、「好好吃飯」、「別熬夜」等。
- **繽紛色彩搭配**：隨機套用多種亮麗且溫和的背景色，呈現出層次豐富的視覺效果。
- **安全退出機制**：綁定鍵盤空白鍵監聽，確保在任何時候都能一鍵關閉程式，避免視窗殘留。

## 自訂配置

您可以直接編輯 [Love-in-Progress.py](file:///d:/Code/Love-in-Progress/Love-in-Progress.py) 檔案來調整問候語與色彩搭配：

### 修改問候語

在程式的第 3 行，您可以修改或增加 `t` 陣列中的文字：

```python
t = ["愛你老幾","好好愛自己","好好吃飯","保持好心情","我想你了","順順利利","別熬夜","天涼了多穿衣服"]
```

### 修改視窗色彩

在程式的第 4 行，您可以自訂 `c` 陣列中的背景色彩名稱或十六進位色碼：

```python
c = ["pink","lightblue","lightgreen","lemonchiffon","hotpink","skyblue"]
```
